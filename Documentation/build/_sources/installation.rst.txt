Installation
============


Requirements
------------

To run QUake-MD GUI, python 3 is needed with the following python libraries:
 * matplotlib 3.5.1
 * basemap 1.2.2
 * numpy 1.21.5
 * pandas 1.4.2
 * scipy 1.7.3

The Anaconda distribution of python includes all these libraries except for the Basemap library.
The open-source Basemap library can be downloaded and installed from the Anaconda cloud.

QUake-MD GUI has been tested with Python 3.9.12 and the python library versions 
previously listed on a windows environment.

Install
-------

Clone the repository on your chosen local repository or download it.
